Overview

Images

When adding an Image to your HTML page you will have code that looks something like below: 


	<h3>The First Avenger>/h3<
	<img src="Images\CaptainAmerica.jpg" alt="Captain America" style="width:304px;height:428px;">
A heading to state what the image is, the image itself, an 'alt' with some descriptive text about the image and the image specifications. So, taking the code from above, on our HTML page it will look like this:

The First Avenger

Links

A link is a....... When adding a link to your HTML page the code will look something like this: 


<a href="https://en.wikipedia.org/wiki/Lego_Super_Heroes"> Lego Super Heroes </a>

Above is the link to the web page you wish your readers to be brought to along side the message the readers will be able to see. Below is what will appear on the screen to your readers.

Lego Super Heroes
