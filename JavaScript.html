While loops

Overview

While loops continue doing a certain function while the condition is true 


				while(somethingHereIsTrue) {
					ThisCodeIsRun
				}
For example


				while(true){
					ThisCodeRunsForever
				}

				while(false){
					ThisCodeNeverRuns
				}
				
				var counter = 0;

				while(counter < 10){
					alert("The value of the counter is: " + counter);
					counter++;
				}
